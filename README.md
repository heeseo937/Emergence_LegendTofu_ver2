# 레전두부 — legendubu.html

> **구급대원 분들을 위한 의사결정보조시스템, 레전두부 입니다.**

현장 구급대원이 즉각적으로 사용할 수 있도록 설계된 **간결하고 직관적인** 응급의료 의사결정 보조 웹 앱입니다.

---

## 개요

| 항목 | 내용 |
|------|------|
| 파일명 | `legendubu.html` |
| 대상 사용자 | 현장 구급대원 |
| 동작 방식 | 단일 HTML 파일 (서버 불필요, 브라우저에서 바로 실행) |
| 진입점 | 랜딩 페이지 → 상황 유형 선택 → 해당 기능 화면 |

## 화면 구성

### 랜딩 페이지
- "레전두부" 브랜드 로고와 서비스 소개 문구 표시
- 두 가지 상황 유형 버튼으로 즉시 진입:

| 버튼 | 설명 |
|------|------|
| 🚨 **일반 응급 상황** | 단일 환자 이송 시 최적 병원 추천 |
| 🚑 **다수 사상자 발생** | MCI 트리아지 보드로 다수 환자 관리 |

- 좌측 상단 `← 홈` 버튼으로 언제든지 랜딩으로 복귀

---

### 화면 1: 일반 응급 상황 (병원 추천)

**입력 조건**

| 항목 | 입력값 |
|------|--------|
| KTAS 등급 | 1(즉각) ~ 5(비응급) |
| 필요 전문과 | 일반·심장·신경·외상·화상·소아·정신·산부인과 |
| 사고 위치 | 포항 북구 / 남구 / 경주 방면 / 영일만 |
| 환자 나이 | 숫자 입력 |
| 기존 진료 병원 | 텍스트 입력 |

**추천 결과**
- 7가지 기준 종합 점수 순으로 병원 카드 나열
  - 중증도 / 전문과 매칭 / 가용 병상 / 당직 현황 / 거리·ETA / 혼잡도 / 기존 진료 기록
- 각 카드에 추천 점수 바, 병상 현황, 전문과 태그, 운영 상태 표시
- 전화번호 클릭 시 바로 전화 연결 (`tel:` 링크)

---

### 화면 2: 다수 사상자 발생 (MCI 관리)

**사고 기본 정보**
- 사고 유형, 발생 위치, 현장 지휘관 입력

**트리아지 보드**

| 존 | 색상 | KTAS |
|----|------|------|
| RED | 🔴 | 1 ~ 2 (즉각 처치) |
| YELLOW | 🟡 | 3 (지연 처치) |
| GREEN | 🟢 | 4 ~ 5 (경증) |

**환자 추가 (+ 환자 추가 버튼)**
- 모달 입력 항목: 성별, 나이, 주 증상, BP, SpO₂, GCS, KTAS 등급
- 입력한 KTAS에 따라 자동으로 해당 존(RED/YELLOW/GREEN)에 카드 생성
- 입력한 정보가 카드에 그대로 표시됨

**최적 배정 실행 (⚡ 버튼)**
- 현재 보드의 모든 환자를 중증도·거리·가용 병상 기반으로 자동 병원 배정
- 존별 배정 우선순위:
  - RED → 포항성모 → 세명기독 → 동국대 → 영남대 (순환)
  - YELLOW → 세명기독 → 포항의료원 → 포항성모 → 동국대 (순환)
  - GREEN → 포항의료원 → 세명기독 (순환)
- 완료 후 토스트 메시지로 총 배정 인원 안내

**병원별 배정 현황 테이블**
- 각 병원의 현재 배정 인원, 가용 병상, 거리, 전문과, 상태 실시간 표시

## 사용 방법

1. `legendubu.html` 파일을 크롬 또는 사파리 브라우저에서 열기
2. 랜딩 페이지에서 상황에 맞는 버튼 클릭
3. 서버·인터넷 연결 불필요 (완전 오프라인 동작)
